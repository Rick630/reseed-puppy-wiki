安装之前

本项目建议使用 Docker 镜像！ 目前我手边只有群晖，所以只能截图群晖的安装方式，但是理论上大差不差，如果有大佬愿意贡献其他系统的安装方式，请直接提交 PR 或者联系我。

  • 群晖NAS如何安装
  • Windows如何安装

# Docker

# 镜像信息

  • 镜像名称：szzhoubanxian/reseed-puppy

  • 镜像地址：https://hub.docker.com/r/szzhoubanxian/reseed-puppy

# Docker CLI

docker run -d \
  --name=reseed-puppy \
  -v /path/to/database:/reseed-puppy-php/database `# 冒号左边请修改为你想在主机上保存数据库的目录` \
  -v /你的种子目录:/你想设置的容器内能见到的目录 `# 种子目录映射，多个目录需要分别映射进来，之后rp工具配置中只需要填写冒号后面的目录` \
  --network host `# 建议使用host模式，这样访问站点接口不容易出问题，程序的默认端口是1919，如果想使用端口映射就将这一行改为-p 1919:1919` \
  --restart unless-stopped \
  szzhoubanxian/reseed-puppy:latest
<br>
## 版本更新

- 只要容器重启即可同步更新到最新版本，暂未开放自定义更新功能
